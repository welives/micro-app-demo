# micro-app-demo

这个案例的笔记可以[在这里查看](https://welives.github.io/blog/front-end/micro-app.html)

## 1、安装依赖

```sh
pnpm bootstrap
```

## 2、运行项目

```sh
pnpm dev
```

访问：`http://localhost:8000`
